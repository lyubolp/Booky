# Translation Type IDs

A list of all translation Type IDs from the table *translation* in the DB.

* ## **0** - Status

  Status is used to represent the current state of a book, which the user has selected.

  List of statuses:

  - **0 :** None
  - **1 :** Reading
  - **2 :** Completed
  - **3 :** On Hold
  - **4 :** Dropped
  - **5 : **Wishlist

* ## **1** - Rank

  Rank is used to give users certain permissions.

  List of ranks:

  - **0 :** Guest
  - **1 :** Rookie
  - **2 :** Reader
  - **3 :** Book Wizard
  - **4 :** Moderator
  - **5 :** Admin
  - **6 :** Author

* ## **2** - Country

* ## **3** - Language

* ## **4** - Author Name

* ## **5** - Author Pen Name

* ## **6** - Tag

* ## **7** - Book Title

* ## **8** - Series title

* ## **9** - Achievement Title

* ## **10** - Achievement Description
